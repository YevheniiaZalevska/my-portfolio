Phonebook

Developed a Phonebook application in React to store contacts. Used state (useState) to manage the contact list and filtering. Implemented validation of input data and uniqueness of contacts. Added styling using CSS and responsive design. Used local storage (localStorage) to store contacts between sessions. A simple phonebook connected to Swagger backend API - you can register, log in, add, remove or filter contacts, log out.
